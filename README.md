# Applied AI Weekly

**Weekly deep dives into applied Agentic and Generative AI.**

Most AI content tells you what happened. This repo is about what you can *do* with it.

Each week covers the most significant breakthroughs in agentic and generative AI — with publication-grade visual abstracts, runnable Google Colab notebooks, and live Hugging Face demos you can interact with in your browser. No setup. No copy-pasting boilerplate. Just open and run.

Written by [Omri Matalon](https://www.linkedin.com/in/omri-matalon), SVP R&D at [QuantHealth](https://www.quanthealth.ai) — at the intersection of medicine, biomedical research, and applied AI engineering.

## How to use the notebooks

Every Colab notebook in this repo can be launched directly — no local installation, no account required beyond a free Google login.

Click the badge inside any week's `README_colab.md`, or use the pattern below:

```
https://colab.research.google.com/github/Omri-Matalon/applied-ai-weekly/blob/main/week-XX/notebook.ipynb
```

Some notebooks use free API keys (e.g., Anthropic, OpenAI free tier). Each notebook lists exactly what's needed at the top of the first cell.

## How to use the HF Space demos

Each week's interactive demo is deployed as a live Hugging Face Space. Links are in the weekly index above and in each week's LinkedIn post. No account needed to use them — just open the link and start experimenting.
